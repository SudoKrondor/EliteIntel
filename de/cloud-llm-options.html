Keine lokale Hardware? Nutze ein Cloud-LLM.

LLM (KI-Gehirn)

Cloud-Option: Gib deinen API-Schlüssel für Mistral, xAI, OpenAI oder Anthropic/Claude ein. Die App verwendet pro Anbieter ein festes Modell:

  • Mistral: 'mistral-small-2506' (Kostenloser Tarif)
  • xAI: grok-4-1-fast-non-reasoning
  • OpenAI: gpt-4.1-mini (Befehle) / gpt-5.2 (Abfragen)
  • Gemini Generative Language API: gemini-3.1-flash-lite-preview für Befehle und Abfragen
  • Anthropic/Claude
  • DeepSeek

Die Kosten variieren je nach gewähltem Cloud-Dienst und Spielzeit.

KOSTENLOSE Cloud-Option: Mistral

  1. Gehe zur Mistral Console
  2. Erstelle ein Konto mit einer gültigen, verifizierbaren E-Mail-Adresse.
  3. KEINE KREDITKARTE ERFORDERLICH
  4. Erstelle eine „Organisation" (beliebiger Name, z. B. „Elite Intel")
  5. Generiere einen API-Schlüssel. Trage diesen in die App ein und starte sie neu.

Option A: xAI API-Schlüssel

  1. Gehe zur xAI Console.
  2. Registriere dich oder melde dich an.
  3. Navigiere zum API-Bereich und generiere einen neuen API-Schlüssel.
  4. Lade dein Konto mit Guthaben auf.
  5. Füge den Schlüssel in das LLM-Feld ein und aktiviere das Schloss-Kontrollkästchen.

Option B: OpenAI API-Schlüssel

  1. Gehe zur OpenAI Platform.
  2. Registriere dich oder melde dich an.
  3. Navigiere zum API-Bereich und generiere einen neuen API-Schlüssel.
  4. Füge den Schlüssel in das LLM-Feld ein und aktiviere das Schloss-Kontrollkästchen.

Option C: Anthropic/Claude API-Schlüssel

  1. Gehe zur Claude Platform.
  2. Melde dich mit E-Mail oder Google an. Hinweis: Die Authentifizierung erfolgt über einen Magic Link, der an deine E-Mail gesendet wird.
  3. Gehe zu Einstellungen → Abrechnung und lade Guthaben auf, bevor du einen Schlüssel erstellst. Ein Schlüssel, der für ein Konto ohne Guthaben erstellt wurde, funktioniert nicht – auch wenn danach Guthaben hinzugefügt wird.
  4. Gehe zu API-Schlüssel und erstelle einen Schlüssel.
  5. Füge ihn in das LLM-Feld ein, aktiviere das Schloss-Kontrollkästchen und starte die Dienste auf dem KI-Tab.

Google TTS-Schlüssel erhalten (14 Stimmen)

  1. Gehe zur Google Cloud Console.
  2. Melde dich an oder erstelle ein Konto.
  3. Erstelle ein neues Projekt.
  4. Aktiviere die Generative Language API für das LLM und/oder die Cloud Text-to-Speech API für TTS.
  5. Gehe zu Anmeldedaten, erstelle einen API-Schlüssel und kopiere ihn.
  6. Schlüssel einschränken: Klicke auf den soeben erstellten Schlüssel. Klicke auf der Detailseite auf Schlüssel einschränken. Ein Dropdown-Menü erscheint. Wähle jede aktivierte API aus (STT und/oder TTS) und klicke auf Speichern.
  7. Füge den Schlüssel in die Felder Spracherkennung und/oder Sprachausgabe in der App ein. Aktiviere die Schloss-Kontrollkästchen.
