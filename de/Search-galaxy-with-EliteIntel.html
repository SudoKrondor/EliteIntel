Suche

Einen Abbauort finden

Eine nützliche Funktion für Weltraum-Miner. Finde, wo ich Low Temperature Diamonds innerhalb von X Lichtjahren abbauen kann Die App fragt Spansh nach einem bekannten Standort eines Sternensystems ab, das einen Ringplaneten mit Vorkommen des angegebenen Materials enthält. Besonders nützlich, wenn Sie und Ihr Flottenträger weit draußen im Schwarzen sind. Sie müssen sich nicht an die Tritium-Autobahn halten. Andere Entdecker haben viele Tritium-Standorte in der gesamten Galaxie gemeldet, und Sie werden mit hoher Wahrscheinlichkeit einen innerhalb von ein oder zwei Trägersprüngen von Ihrer Position finden.

Eine Ware finden

Um eine bestimmte Ware zu finden, verwenden Sie den folgenden Ablauf:

Eine spezifische Warensuche anfordern:

Sprechen Sie natürlich mit EliteIntel. Zum Beispiel: Wo kann ich Tritium kaufen? oder Finde einen Markt, der Bromellite hat. EliteIntel fragt Community-Daten nach den besten Marktpreisen innerhalb von X Lichtjahren Ihrer aktuellen Position ab. Wenn Ergebnisse gefunden werden, wird eine Route zur besten Übereinstimmung geplant. Sie können Ihre Suche verfeinern, indem Sie nach dem günstigsten oder nächstgelegenen fragen und eine Reichweite angeben. Zum Beispiel: Finde einen Markt, auf dem wir Gold innerhalb von 50 Lichtjahren kaufen können, nimm den günstigsten

HINWEIS: Die Marktdaten sind zeitkritisch. Wenn keine Commander mit EDMC eine Station passiert haben, weder EDSM noch Spansh aktuelle Informationen haben. Selbst wenn Sie es in INARA sehen. INARA hat keine API, über die Apps Abfragen stellen können, daher können wir dort keine Informationen abrufen.

Einen Händler / Vermittler finden

Fragen Sie die App: Finde einen Händler für Roh-/hergestellte/codierte Materialien innerhalb von X Lichtjahren. Sie durchsucht Spansh und plant eine Route zum nächstgelegenen Sternensystem, in dem sich ein Materialshändler befindet, gemäß den Daten in Spansh.

Sie können fragen: Finde einen Guardian/Human-Technologiehändler innerhalb von X Lichtjahren. Wenn Sie keine Reichweite angeben, wird innerhalb von etwa 100 Lj von Ihrem Standort gesucht. (kann sich in der Zukunft auf X mal Ihre Schiffsreichweite ändern)

HINWEIS: Wie oben. Die Daten, die wir erhalten, können veraltet sein. Die App teilt Ihnen mit, wie lange es her ist, dass die Daten aktualisiert wurden, und wenn das Ergebnis einen Monat alt ist, hat dieses System möglicherweise nicht mehr den gesuchten Händler.
