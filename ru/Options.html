Параметры интерфейса и настройки

Вкладка ИИ

Это главная вкладка по умолчанию.
- Запустить / Остановить службы: Включает или выключает стек ИИ.
- Бодрствование/Сон: В режиме бодрствования приложение слушает постоянно. В режиме сна приложение игнорирует ввод, если не нажата кнопка PTT, не использовано слово обхода "Listen" или не подана команда "Wake up!".
- Оверлей OBS: Отображает чёрное оверлейное окно с взаимодействием Командира и ИИ. Добавьте в OBS, убрав чёрный фон.
- Аудиоустройства: Выбор устройства для ввода/вывода звука. Калибровка звука: Запуск калибровки для лучшей производительности.

Вкладка Игрок

  • Имя Командира: Используйте это поле для замены вашего внутриигрового имени при синтезе речи.
  • Параметры корабля: Эти автоматизации можно включать и отключать. Полезно для командиров с ограниченными возможностями.
  • Управление флотом: Назначайте голоса, личности и темп отдельным кораблям. Личности работают только с облачными LLM. Значок шестерёнки открывает свойства корабля: автоматический гонк и торговый профиль.

  • Автоматический гонк при входе в систему: Выберите группу огня и триггер. Если эта опция отмечена, корабль будет выполнять сканирование при входе. Если HUD в режиме боя, он переключится в режим анализа, выполнит сканирование и вернётся обратно.
  • Настроить торговый профиль: Эти параметры можно задать через интерфейс или голосовыми командами: "alter/change trade profile set [параметр] to [значение]"

Вкладка Действия

Вкладка Действия / Привязки содержит три раздела: Привязки, Встроенные команды и Пользовательские команды.

  • Привязки: каталог, где находится файл привязок управления игрой. Без него приложение не может управлять игрой.
  • Профиль: ваш текущий профиль привязок в игре.
  • Файл: файл, содержащий используемые привязки.

Вы можете изменить привязки на этом экране и сохранить их как новый профиль.

ПРИМЕЧАНИЕ HOTAS/КОНТРОЛЛЕРЫ отображаются, но не могут быть настроены через этот экран. Только привязки клавиатуры (может измениться в будущем).

Действия / Встроенные команды

Отображает список встроенных команд. Двойной щелчок по команде откроет диалоговое окно с информацией о ней и позволит предложить более качественный перевод для локализации.

Пользовательские команды

Этот экран позволяет задать пользовательское действие, которое приложение выполнит по вашей команде.

  • Нажмите кнопку НОВЫЙ, чтобы открыть всплывающее окно, в котором можно определить пользовательское действие.

  • Введите название действия. ПРИМЕЧАНИЕ: название должно содержать слова (токены), разделённые символом подчёркивания _
  • Укажите имя вашего пользовательского действия.
  • Укажите описание вашего пользовательского действия.
  • Введите обучающие слова это смысловые токены. LLM попытается сопоставить произнесённую команду с действием, используя наибольшую вероятность. Чем точнее ваши токены описывают действие, тем выше вероятность его распознавания.

Для использования пользовательских действий говорите естественно запоминать точные слова не нужно, но необходимо передать точный смысл, чтобы LLM сопоставил команду с нужным действием с наибольшей вероятностью.
