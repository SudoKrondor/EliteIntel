Нет локального железа? Используй облачный LLM.

LLM (мозг ИИ)

Облачный вариант: Введи свой API-ключ для Mistral, xAI, OpenAI или Anthropic/Claude. Приложение использует фиксированную модель для каждого провайдера:

  • Mistral: 'mistral-small-2506' (Бесплатный уровень)
  • xAI: grok-4-1-fast-non-reasoning
  • OpenAI: gpt-4.1-mini (команды) / gpt-5.2 (запросы)
  • Gemini Generative Language API: gemini-3.1-flash-lite-preview для команд и запросов
  • Anthropic/Claude
  • DeepSeek

Стоимость будет зависеть от выбранного облачного сервиса и продолжительности игровых сессий.

БЕСПЛАТНЫЙ облачный вариант: Mistral

  1. Перейди в консоль Mistral
  2. Создай аккаунт с действующим адресом электронной почты, который можно подтвердить.
  3. БАНКОВСКАЯ КАРТА НЕ НУЖНА
  4. Создай «Организацию» (назови как угодно, например «Elite Intel»)
  5. Сгенерируй API-ключ. Введи его в приложение и перезапусти его.

Вариант А: API-ключ xAI

  1. Перейди в консоль xAI.
  2. Зарегистрируйся или войди в аккаунт.
  3. Перейди в раздел API и сгенерируй новый API-ключ.
  4. Пополни баланс аккаунта.
  5. Вставь ключ в поле LLM и установи флажок блокировки.

Вариант Б: API-ключ OpenAI

  1. Перейди на платформу OpenAI.
  2. Зарегистрируйся или войди в аккаунт.
  3. Перейди в раздел API и сгенерируй новый API-ключ.
  4. Вставь ключ в поле LLM и установи флажок блокировки.

Вариант В: API-ключ Anthropic/Claude

  1. Перейди на платформу Claude.
  2. Войди через электронную почту или Google. Примечание: аутентификация выполняется через магическую ссылку, отправленную на почту.
  3. Перейди в Настройки → Платежи и пополни баланс, прежде чем создавать ключ. Ключ, созданный на аккаунте без средств, не работает, даже если средства будут добавлены позже.
  4. Перейди в API-ключи и создай ключ.
  5. Вставь его в поле LLM, установи флажок блокировки и запусти или перезапусти сервисы на вкладке ИИ.

Получение ключа Google TTS (14 голосов)

  1. Перейди в консоль Google Cloud.
  2. Войди или создай аккаунт.
  3. Создай новый проект.
  4. Включи Generative Language API для LLM и/или Cloud Text-to-Speech API для TTS.
  5. Перейди в Учётные данные, создай API-ключ и скопируй его.
  6. Ограничь ключ: нажми на только что созданный ключ. На странице сведений нажми Ограничить ключ. Появится выпадающий список. Отметь каждый API, который ты включил (STT и/или TTS), и нажми Сохранить.
  7. Вставь ключ в поля Распознавание речи и/или Синтез речи в приложении. Установи флажки блокировки.
