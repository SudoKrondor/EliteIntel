Linux Windows

  1. Скачайте 👉установщик👈.
  2. Запустите установщик и следуйте инструкциям на экране.
  3. Настройте LLM. Доступны два варианта:
    • Локальный LLM (бесплатно, офлайн): см. руководство Локальный LLM. Требуется мощное GPU-оборудование.
    • Облачный LLM (есть бесплатный вариант, проще в настройке): см. руководство Настройка приложения для настройки ключа API.
