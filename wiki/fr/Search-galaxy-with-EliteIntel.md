# Recherche dans la galaxie

## Trouver un site minier

Une fonctionnalité très utile pour les mineurs spatiaux. ``Trouve où je peux miner des Diamants basse température dans un rayon de X années-lumière``
L'application interrogera Spansh pour trouver un système stellaire connu contenant une planète avec anneaux comportant des gisements du
matériau spécifié. Particulièrement utile lorsque vous et votre Fleet Carrier vous trouvez loin dans le vide interstellaire. Vous n'avez pas 
à vous en tenir à la route du tritium. D'autres explorateurs ont signalé de nombreux sites de tritium partout dans la galaxie, et vous en trouverez un
dans un ou deux sauts de carrier depuis votre position avec une forte probabilité.


## Trouver une marchandise

Pour localiser une marchandise spécifique, procédez comme suit :

**Demandez une recherche de marchandise spécifique** : 

Parlez naturellement à EliteIntel. Par exemple, ``Où puis-je acheter du Tritium ?`` 
ou ``Trouve un marché qui a de la Bromellite.`` EliteIntel interroge les données participatives pour les meilleurs prix de marché 
dans un rayon de X années-lumière de votre position actuelle. Lorsque des résultats sont trouvés, il tracera une route vers la
meilleure correspondance. Vous pouvez affiner vos requêtes en demandant le moins cher ou le plus proche, et indiquer une portée. Par exemple :
``Trouve un marché où l'on peut acheter de l'or dans un rayon de 50 années-lumière, prends le moins cher``

NOTE :
_Les données de marché sont sensibles au temps. Si aucun commandant utilisant EDMC n'est passé par une station, ni EDSM 
ni Spansh n'auront d'informations à jour. Même si vous les voyez dans INARA. INARA ne dispose pas d'une API 
permettant aux applications de l'interroger, donc nous ne pouvons pas obtenir d'informations de là._


## Trouver un courtier / négociant

Demandez à l'application ``Trouve un négociant en matériaux bruts/manufacturés/encodés dans un rayon de X années-lumière``. Elle recherchera dans Spansh et tracera une route
vers le système stellaire le plus proche où se trouve un négociant en matériaux, selon les données de Spansh.

Vous pouvez demander ``Localise un courtier technologique Guardian/Humain dans un rayon de X années-lumière``. Si vous n'indiquez pas la portée,
l'application recherchera dans un rayon d'environ 100 al depuis votre position. (susceptible d'évoluer dans le futur vers X fois la portée de saut de votre vaisseau)

NOTE :
_Comme ci-dessus. Les données que nous obtenons peuvent être anciennes. L'application vous indiquera depuis combien de temps les données ont été mises à jour, et
si le résultat date d'il y a un mois, il se peut que ce système n'ait plus le courtier que vous cherchez._
