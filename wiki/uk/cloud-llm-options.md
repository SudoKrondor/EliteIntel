
# Немає локального заліза? Скористайся хмарним LLM.


**LLM (мозок ШІ)**

*Хмарний варіант:* Введи свій API-ключ для Mistral, xAI, OpenAI або Anthropic/Claude. Застосунок використовує фіксовану модель для кожного провайдера:
- **Mistral**: 'mistral-small-2506' **(Безкоштовний рівень)**
- **xAI**: `grok-4-1-fast-non-reasoning`
- **OpenAI**: `gpt-4.1-mini` (команди) / `gpt-5.2` (запити)
- **Gemini Generative Language API**: `gemini-3.1-flash-lite-preview` для команд і запитів
- **Anthropic/Claude**
- **DeepSeek**


Вартість залежатиме від обраного хмарного сервісу та тривалості гри.

### БЕЗКОШТОВНИЙ хмарний варіант: Mistral
1. Перейди до [консолі Mistral](https://console.mistral.ai/home)
2. Створи обліковий запис із дійсною електронною адресою, яку можна підтвердити.
3. КРЕДИТНА КАРТКА НЕ ПОТРІБНА
4. Створи «Організацію» (назви як завгодно, наприклад «Elite Intel»)
5. Згенеруй API-ключ. Введи його в застосунок і перезапусти його.


### Варіант А: API-ключ xAI
1. Перейди до [консолі xAI](https://console.x.ai/).
2. Зареєструйся або увійди в систему.
3. Перейди до розділу API та згенеруй новий API-ключ.
4. Поповни рахунок у своєму обліковому записі.
5. Встав ключ у поле **LLM** і встанови прапорець блокування.

### Варіант Б: API-ключ OpenAI
1. Перейди до [платформи OpenAI](https://platform.openai.com/).
2. Зареєструйся або увійди в систему.
3. Перейди до розділу API та згенеруй новий API-ключ.
4. Встав ключ у поле **LLM** і встанови прапорець блокування.

### Варіант В: API-ключ Anthropic/Claude
1. Перейди до [платформи Claude](https://platform.claude.com).
2. Увійди через електронну пошту або Google. Примітка: автентифікація здійснюється за допомогою чарівного посилання, надісланого на твою пошту.
3. Перейди до **Налаштування → Виставлення рахунків** і поповни рахунок, перш ніж створювати ключ. Ключ, створений на незадіяному обліковому записі, не працюватиме навіть після поповнення.
4. Перейди до **API-ключів** і створи ключ.
5. Встав його в поле **LLM**, встанови прапорець блокування та запусти або перезапусти служби на вкладці ШІ.

### Отримання ключа Google TTS (14 голосів)

1. Перейди до [консолі Google Cloud](https://console.cloud.google.com/).
2. Увійди або створи обліковий запис.
3. Створи новий проєкт.
4. Увімкни **Generative Language API** для LLM та/або **Cloud Text-to-Speech API** для TTS.
5. Перейди до **Облікові дані**, створи API-ключ і скопіюй його.
6. **Обмеж ключ**: Натисни на щойно створений ключ. На сторінці деталей натисни **Обмежити ключ**. З'явиться випадаючий список. Відмітьте кожен API, який ти увімкнув (STT та/або TTS), і натисни **Зберегти**.
7. Встав ключ у поля **Розпізнавання мовлення** та/або **Синтез мовлення** в застосунку. Встанови прапорці блокування.
