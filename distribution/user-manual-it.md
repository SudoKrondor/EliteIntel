# Guida ai Comandi e alle Query di EliteIntel (Italiano)

Ehi Comandante! Questa è una guida di riferimento per i tipi di cose che puoi chiedere o dire al tuo assistente **Elite Intel**.

L'obiettivo non è ricordare a memoria gli esempi, puoi parlare in modo naturale: mostrano semplicemente il tipo di richieste che l'assistente sa comprendere.

## Prima di tutto. Hai dei problemi?

- Se l'app esegue comandi a caso senza motivo: problema di cervello (LLM non adatta allo scopo o configurazione sbagliata)
- Se l'app esegue comandi ma non si attivano: problema di mani (tasti di scelta rapida non configurati)
- Se l'app fatica a sentirti: problema di orecchie (stanza rumorosa, rapporto rumore/RMS basso, audio non calibrato, ecc.)
- Se l'app non parla: problema di bocca (controlla il routing audio a livello di sistema operativo)

### [La pagina Wiki completa è qui](https://www.elite-intel.org/it)

## Input Audio

Calibra l'audio nell'app. Se la differenza tra il Noise Floor e l'RMS è troppo bassa, ad esempio meno di 400, l'app avrà difficoltà a capirti.

Un buon rapporto è almeno 800-1000. Se usi altoparlanti e microfono, abilita il **Push-to-Talk**; altrimenti è consigliabile usare cuffie con microfono.

## App

- Check missing or unbound key bindings.
- Sleep / Wake Up (Ignores user voice input in sleep mode, by pass by saying 'Listen Up', or switch back by saying 'Wake up')

## 💬 Modalità Conversazione

Per impostazione predefinita Elite Intel funziona in **Modalità Strict**: se l'input non corrisponde a un comando o query noto, viene silenziosamente ignorato. È intenzionale — evita che il rumore STT e le chiacchiere di sottofondo attivino azioni casuali durante il volo.

Abilita la **Modalità Conversazione** nelle impostazioni per la chat libera. In questa modalità, tutto ciò che non corrisponde a un comando diventa conversazione generale — dettagli del gioco, argomenti del mondo reale, configurazioni nave, qualsiasi cosa.

Gli LLM locali risponderanno ma saranno più formali. Gli LLM cloud (Claude, OpenAI, xAI, Mistral, Deepseek) sono consigliati per una conversazione più naturale.
