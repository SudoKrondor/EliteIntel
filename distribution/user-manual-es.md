# Guía de consultas y comandos de IA – EliteIntel (Español)

¡Hola, Comandante! Esta es una referencia de los tipos de cosas que puedes preguntarle o decirle a tu asistente **Elite Intel**.
**Lo ideal es que no necesites memorizar nada de esto** — habla con naturalidad y la aplicación descifra lo que quieres decir. Esta lista existe para que sepas qué es posible, no para que recites guiones.

## Lo primero es lo primero. ¿Tienes problemas?

- Si la aplicación ejecuta comandos aleatorios sin razón: problema de cerebro (LLM débil o configuración incorrecta)
- Si la aplicación ejecuta comandos pero no se activan: problema de manos (asignaciones de teclas)
- Si la aplicación no te escucha bien: problema de oídos (sala ruidosa, ratio ruido/RMS bajo, audio sin calibrar, etc.)
- Si la aplicación no habla: problema de boca. Verifica el enrutamiento de audio a nivel del sistema operativo.

### [Wiki completo aquí](https://www.elite-intel.org/es)

## Entrada de audio

**Calibra el audio en la aplicación.** Si la diferencia entre el nivel de ruido y el RMS es demasiado baja (p. ej. menos de 400),
la aplicación tendrá dificultades para entenderte. Un buen ratio es al menos 800–1000.
Si utilizas altavoces y micrófono, activa la función «pulsar para hablar»; de lo contrario, se recomienda usar auriculares con micrófono.

## 💬 Chat general (el modo Conversación debe estar ACTIVADO)

Por defecto, la aplicación funciona en **Modo Estricto**: si la entrada no coincide con un comando o consulta conocida, se ignora en silencio. Esto es intencional — evita que el ruido del STT y las conversaciones de fondo activen acciones aleatorias durante el vuelo.

Activa el **Modo Conversación** en la pestaña de Configuración para habilitar el chat libre. Una vez activado, todo lo que no coincida con un comando deriva a conversación general — lore del juego, temas del mundo real, configuraciones de naves, lo que sea.

Los LLMs locales responderán pero serán rígidos. Se recomiendan LLMs en la nube (Claude, OpenAI, xAI, Mistral, Deepseek) para conversaciones.
