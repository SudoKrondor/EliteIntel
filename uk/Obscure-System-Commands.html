Маловідомі системні команди

EliteIntel має мінімалістичний інтерфейс без меню або складних панелей налаштувань. Повний набір функцій доступний через голосові команди. Усі налаштування та функції доступні через природне мовлення.

Перемикані налаштування

Перемикайте ці налаштування за допомогою голосових команд:

  • Route Announcements [On/Off]: «Turn Route Announcements On.» EliteIntel оголошує деталі про ваш наступний стрибок, зокрема чи придатна зірка для заправки, безпека системи та іншу відповідну інформацію. При вимкненні оголошення зупиняються. Ручні запити залишаються доступними. За замовчуванням: увімк.
  • Discovery Announcements [On/Off]: «Turn Discovery Announcements On.» EliteIntel оголошує системи першовідкривача, цінні планети, біологічні сигнали та подібні відкриття. За замовчуванням: увімк.
  • Mining and Material Announcements [On/Off]: «Turn Mining Announcements On.» EliteIntel позначає удари проспекторних лімпетів та знахідки матеріалів. За замовчуванням: вимк.
  • Add Mining Target [Material Name]: «Add Mining Target Painite.» EliteIntel позначає удари для цього конкретного матеріалу. Немає ефекту, якщо ціль не встановлена. За замовчуванням: вимк.
  • Radio Chatter [On/Off]: «Turn On the Radio.» EliteIntel оголошує відповідні радіопередачі, наприклад піратські погрози, відфільтровуючи рутинний трафік, як комерційні трансляції. Корисно для вантажних рейсів. За замовчуванням: вимк.
  • Streaming Mode [On/Off]: «Set Streaming Mode On.» EliteIntel відповідає лише на команди з префіксом «Computer.» Корисно для стрімінгу або при грі у крилі. Скажіть «Computer, turn streaming mode off» для вимкнення.
  • Night Vision [On/Off]: «Night vision on.» Вмикає/вимикає нічний зір у кораблі або SRV.
  • Headlights [On/Off]: «Headlights on.»
  • Drive Assist [On/Off]: «Drive assist off.»

Навігація та пошук

Крім базового прокладання маршрутів, EliteIntel підтримує цільову навігацію:

  • Navigate to Coordinates: «Navigate to latitude 41.4325 longitude -75.2309.» EliteIntel проведе вас з орбіти до 50 метрів. Навігація на темному боці потребує приладового польоту.
  • Navigate to Bio Sample / Codex Entry: «Navigate to next bio sample» або «Navigate to nearest codex entry.» EliteIntel переходить до найближчого позначеного біологічного об'єкта. Пріоритизує активний рід, якщо він відстежується.
  • Navigate to Landing Zone: «Navigate to landing zone.» Переходить до останніх відомих координат посадки на землі.
  • Navigate to Fleet Carrier: «Navigate to carrier» або «Return to base.» Прокладає маршрут до останньої відомої системи носія.
  • Take Me Home: «Take me home.» Прокладає маршрут до вашої налаштованої домашньої системи.
  • Set Home System: «Set as home system.» Позначає вашу поточну систему як домашню.
  • Plot Fleet Carrier Route: Відкрийте карту галактики та виберіть зірку. Скопіюйте її назву за допомогою останньої кнопки праворуч в інтерфейсі карти галактики. Скажіть «Calculate Fleet Carrier Route.» EliteIntel будує маршрут, якщо система є в Spansh.
  • Enter Next Fleet Carrier Destination: Відкрийте карту галактики флотського носія та натисніть верхнє текстове поле. Скажіть «Enter next Fleet Carrier destination.» EliteIntel вставляє назву наступної системи у поле.
  • Find Nearest Human/Guardian Tech Broker: «Find nearest Human tech broker.» EliteIntel прокладає маршрут. Після прибуття скажіть «Remind me where we need to go» для нагадування про станцію.
  • Find Nearest Material Trader: «Find nearest raw material trader» / «encoded» / «manufactured.» EliteIntel знаходить і прокладає маршрут.
  • Find Brain Trees: «Find brain trees within 500 light years.» EliteIntel розвідує місця розташування захисної біології Стражів.
  • Find Mining Location: «Find where we can mine some Osmium within 200 light years.» EliteIntel знаходить систему з відповідними кільцями та прокладає маршрут.
  • Find Carrier Fuel (Tritium): «Find where we can mine some carrier fuel within 300 light years.» EliteIntel шукає кільцеві системи, що містять тритій.
  • Find Commodity: «Find where we can buy Bromellite within 150 light years.» EliteIntel запитує EDSM та прокладає маршрут до найкращого ринку.
  • Find Nearest Vista Genomics: «Find nearest Vista Genomics.» Прокладає маршрут до найближчого Vista Genomics для здачі біологічних зразків.
  • Find Nearest Fleet Carrier: «Find nearest Fleet Carrier.» Корисно, коли поблизу потрібен флотський носій.

Команди бою та місій

  • Find Hunting Grounds: «Find hunting grounds.» EliteIntel розвідує найближчі небезпечні зони видобутку ресурсів або зони бою, що відповідають параметрам місій.
  • Plot Reconnaissance Route: «Navigate / plot recon route to hunting ground.» EliteIntel прокладає розвідувальний шлях до цільової системи.
  • Navigate to Mission Provider: «Navigate to system with matching mission provider» або «Navigate to a confirmed pirate massacre mission provider.» EliteIntel прокладає маршрут до постачальника місій.
  • Navigate to Active Mission: «Navigate to active mission.» Навігація до поточної цілі місії.
  • Confirm / Ignore Hunting Ground: «Confirm hunting ground» або «Ignore hunting ground.» Повідомляє EliteIntel, чи використовувати або пропустити це місце.

Скорочення управління кораблем

  • Power Distribution: «All power to shields,» «engines,» «weapons,» або «Equalize power.» Одна команда перебалансовує всі піпси живлення. Особливо корисно у VR, де затримки хмарної обробки збільшують час відповіді.
  • Exit to HUD: «Exit to HUD.» Виходить із вкладених меню однією командою.
  • FSS Scan / Honk: «Open FSS and scan» або «Honk.» Запускає сканування системи.
  • Set Optimal Speed: «Set optimal speed.» Встановлює газ на 75% для надсвітлового підходу. Видайте цю команду приблизно за 20 секунд до цілі, щоб уникнути орбітального зациклення.
  • Target Next System in Route: «Target next system in route.» EliteIntel вибирає наступну точку на прокладеному маршруті.
  • Wing Nav Lock: «Wing nav lock.» Блокує навігацію на ведомого у крилі.
  • Target Subsystem: «Target power plant.» Блокує прицілювання на підсистему.
  • Dismiss Ship / Get Extracted: «Dismiss ship» / «Go to orbit» / «Return to surface, requesting extraction.»

Утилітарні команди та команди сесії

  • Set a Reminder: «Set reminder, pick up Painite from Hutton Orbital.» EliteIntel зберігає нагадування для сесії. Скажіть «Remind me» для його отримання.
  • Clear Reminders: «Clear reminders.» Очищає чергу нагадувань.
  • Clear Codex Entries: «Clear codex entries.» Очищає всі відскановані біологічні записи. Використовуйте обережно.
  • Delete Codex Entry: «Delete codex entry.» Видаляє останній запис.
  • Clear Cache: «Clear cache.» Очищає весь кеш сесії. Використовуйте лише тоді, коли сесія перебуває у невідновлюваному стані.
  • Monetize Route: «Monetize route.» EliteIntel аналізує прокладений маршрут на предмет можливостей торгового або дослідницького прибутку.
  • Verify LLM Connection: «Verify LLM connection.» Підтверджує, що бекенд ШІ доступний і відповідає.
  • Interrupt / Silence: «Interrupt,» «Silence,» або «Cancel.» Зупиняє активний вивід синтезу мовлення.
  • Biome Analysis: «Run biome analysis on [star system / planet name].» EliteIntel повідомляє про ймовірні види в цьому місці перед посадкою.
  • Fleet Carrier Financials: «Open fleet carrier data and tell me how long we can operate.» EliteIntel розраховує кількість тижнів роботи від поточного балансу.
  • Fleet Carrier Range: «What is the range of our fleet carrier?» EliteIntel розраховує найгірший радіус дії з тритієм у депо та сховищі носія.
  • How Long Can the Carrier Operate?: «Use reserve balance, how long can we run the carrier with 31 million credits per week?» Вкажіть, який баланс використовувати. EliteIntel виконує розрахунок.

Примітки щодо використання

  • Природна мова: Формальний синтаксис команд не потрібен. Підтримується природне мовлення.
  • Користувачі VR: Команди розподілу живлення та «Exit to HUD» зменшують вплив затримок хмарної обробки у VR.
  • Навігація на темному боці: Навігація до координат на темному боці планети потребує приладового польоту.
