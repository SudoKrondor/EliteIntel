Інтерфейс та параметри конфігурації

Вкладка AI

Це головна/стандартна вкладка.
- Start / Stop Services: Вмикає/вимикає стек ШІ.
- Wake/Sleep: У режимі пробудження застосунок постійно слухає, у режимі сну ігнорує ввід, якщо не натиснута кнопка PTT, не використано обхідне слово «Listen» або не видано команду «Wake up!».
- OBS Overlay: Відображає чорне накладне вікно з взаємодією Командир / ШІ. Додайте до OBS, встановіть ключ на чорному фоні.
- Audio Devices: Виберіть аудіопристрій для введення/виведення. Calibrate Audio: Запустіть калібрування аудіо для кращої продуктивності.

Вкладка Player

  • Commander Name: Використовуйте це поле для заміни свого внутрішньоігрового імені для синтезу мовлення.
  • Ship Options: Ви можете вмикати та вимикати ці автоматизації. Корисно для командирів з обмеженими можливостями.
  • Fleet Management: Призначайте голоси, особистості та темп окремим кораблям. Особистість працює лише з хмарними LLM. Значок шестерні відкриває властивості корабля, такі як авто-сканування та торговий профіль.

  • Honk System On Entry: Виберіть групу вогню та тригер. Якщо цей параметр увімкнено, корабель виконає сканування системи при вході. Якщо ваш HUD перебуває в режимі Combat, він перейде до Analysis, виконає сканування і повернеться назад.
  • Customize your trade profile: Ці параметри можна встановити в інтерфейсі користувача або голосовими командами: «alter/change trade profile set [parameter] to [value]»

Вкладка Actions

Вкладка Actions / Bindings має три розділи: Bindings, Built-in Commands і Custom commands.

  • Bindings каталог, де знаходиться файл прив'язок вашої гри. Без нього застосунок не може керувати грою.
  • Profile ваш поточний профіль прив'язок у грі.
  • File файл, що містить прив'язки, які ви зараз використовуєте.

Ви можете змінювати прив'язки за допомогою цього екрана та зберегти як новий профіль.

ПРИМІТКА: HOTAS/КОНТРОЛЕРИ відображаються, але не можуть бути налаштовані через цей екран. Лише прив'язки клавіатури (може змінитися в майбутньому).

Actions / Built-In Commands

Надає список вбудованих команд. Подвійне клацання на одній з них відкриє діалогове вікно з інформацією про команду і дозволить запропонувати кращий переклад для локалізації.

Custom Commands

Цей екран дозволяє визначити власну дію, яку застосунок виконуватиме за вашою командою.

  • Натисніть кнопку NEW відкриється спливаюче вікно, де можна визначити власну дію.

  • Введіть назву дії. ПРИМІТКА: Назва дії повинна містити слова (токени), розділені підкресленням _
  • Надайте назву для вашої власної дії
  • Надайте опис вашої власної дії
  • Введіть тренувальні слова це смислові токени. LLM намагатиметься зіставити вимовлену команду з дією за найвищою ймовірністю. Чим більш ймовірні ваші токени для відповідності дії, тим частіше вона повертатиметься.

Щоб використовувати власні дії, говоріть природно вам не потрібно запам'ятовувати точні слова, але ви повинні передати точний сенс, щоб LLM зіставив його з вашою командою з найвищою ймовірністю.
