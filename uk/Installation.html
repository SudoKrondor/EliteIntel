Версія V1.0, що вийшла, відрізняється від того, що ви бачите на скріншотах.

Якщо ви хочете версію V1.1, приєднуйтесь до команди бета-тестувальників.

👉Приєднатись до команди бета-тестувальників V1.1 тут👈

Windows

  1. Завантажте 👉інсталятор👈.
  2. Запустіть інсталятор і дотримуйтесь підказок на екрані.
    • Parakeet STT (локальне розпізнавання мовлення) та Kokoro TTS (локальний синтез мовлення) обидва включені. Жодних додаткових кроків або сервісів не потрібно.
  3. Налаштуйте LLM. Доступні два варіанти:
    • Локальний LLM (безкоштовно, офлайн): Дивіться посібник з локального LLM. Потрібна потужна відеокарта.
    • Хмарний LLM (простіше в налаштуванні): Дивіться посібник Налаштування застосунку для налаштування API-ключа.

Linux

Встановлення (будь-який дистрибутив із робочим столом без sudo)

  1. Завантажте скрипт інсталятора:
curl -L -o installer.sh https://raw.githubusercontent.com/stone-alex/EliteIntel/refs/heads/master/distribution/installer.sh
  1. Зробіть скрипт виконуваним і запустіть його:
chmod +x installer.sh
./installer.sh

Застосунок встановлюється в ~/.var/app/elite.intel.app. Обидва Parakeet STT і Kokoro TTS входять до складу застосунку. Додаткове встановлення не потрібне. Увімкніть їх у застосунку через прапорці Settings tab ☑ Use.

  1. Налаштуйте LLM. Доступні два варіанти:
    • Локальний LLM (безкоштовно, офлайн): Дивіться посібник з локального LLM. Потрібна потужна відеокарта.
    • Хмарний LLM (простіше в налаштуванні): Дивіться посібник Налаштування застосунку для налаштування API-ключа.

Налаштування завершено. Дивіться Налаштування застосунку для наступних кроків.

Видалення

Використовуйте прапорець -d для видалення застосунку. Інсталятор запитає підтвердження перед видаленням конфігурації та даних API-ключів.

bash installer.sh -d
