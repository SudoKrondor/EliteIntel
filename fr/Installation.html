Linux Windows

  1. Téléchargez le 👉programme d'installation👈.
  2. Exécutez le programme d'installation et suivez les instructions à l'écran.
  3. Configurez un LLM. Deux options sont disponibles :
    • LLM local (gratuit, hors ligne) : Consultez le guide LLM local. Nécessite un GPU capable.
    • LLM cloud (a une option gratuite et est plus facile à configurer) : Consultez le guide Configurer l'application pour la configuration de la clé API.
