To add custom button names to your device
 - First make sure that your device is mapped in the DeviceMapping.xml
 - Navigate to your control bindings folder
	- on windows this can be found here %localappdata%\Frontier Developments\Elite Dangerous\Options\Bindings
 - Navigate or create the folder DeviceButtonMaps in the control bindings folder
 - Create a file using your device name as the file name with extension of buttonMap. e.g. VPCPanel.buttonMap for the device VPCPanel
 - This is read as an xml file with the root node being called <Root>, then you can remap a button name by adding a tag with the base button name with the data being the new name you want it to display with
	
Example:
<Root>
	<Joy_1>B1</Joy_1>
	<Joy_XAxis>Throttle</Joy_XAxis>
</Root>
This will rename the button Joy_1 to B1 and the axis Joy_XAxis to Throttle


You can also use the existing input icons as well
<Root>
	<Joy_1>[x360A]</Joy_1>
</Root>
This will make the button Joy_1 show the icon for the xbox360 controller A button

Note any changes to these files can be applied by re-entering the control binding screen, no need to restart the game.
